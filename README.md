# 呼吸卡片谷歌插件

## 项目简介

这个项目是一个谷歌浏览器插件，旨在帮助用户进行呼吸练习。通过简单的界面和易于使用的功能，用户可以在浏览器中轻松进行呼吸训练。

## 文件结构

- `background.js`: 插件的后台脚本。
- `content.js`: 插件的内容脚本。
- `images/`: 存储插件图标的文件夹。
- `manifest.json`: 插件的配置文件。
- `popup.html`: 插件的弹出页面。
- `popup.js`: 弹出页面的脚本。

## 使用方法

1. 克隆项目到本地。
2. 在谷歌浏览器中打开扩展程序页面。
3. 加载已解压的扩展程序。
